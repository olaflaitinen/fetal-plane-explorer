# Support

## How to get help

### Questions and Discussions

If you have questions about using Fetal Plane Explorer, please use the [GitHub Discussions](https://github.com/olaflaitinen/fetal-plane-explorer/discussions) (if enabled) or check existing [Issues](https://github.com/olaflaitinen/fetal-plane-explorer/issues).

### Reporting Bugs

If you believe you have found a bug, please check the existing issues to see if it has already been reported. If not, please open a new [Bug Report](https://github.com/olaflaitinen/fetal-plane-explorer/issues/new?template=bug_report.yml).

### Feature Requests

We welcome ideas for new features! Please open a [Feature Request](https://github.com/olaflaitinen/fetal-plane-explorer/issues/new?template=feature_request.yml).

## Scope of Support

This is an open-source research demonstration.
-   **In Scope:** Installation issues, bugs in the provided code, feature requests for the demo logic.
-   **Out of Scope:** Clinical support, diagnosis assistance, integration into electronic health record (EHR) systems, or support for proprietary ultrasound machine formats not standardly supported by image libraries.
