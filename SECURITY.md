## Reporting a Vulnerability

We take security vulnerabilities seriously.

If you discover a security vulnerability within this project, please **DO NOT** create a public GitHub issue. Instead, please describe the vulnerability and send it to the maintainers via a private channel (e.g., email to the repository owner) or use GitHub's private vulnerability reporting feature if enabled.

We will acknowledge receipt of your report within 5 business days and strive to provide a fix in a timely manner.

## Privacy and Clinical Safety

This project is a research demonstration and is **not intended for clinical use**. It processes images but does not persist them by default.
-   Do not upload Protected Health Information (PHI) to public instances.
-   When running locally, ensure your environment is secure.

See the README for full disclaimers.
